NiFi Exporter

Version 1.0.0
A Prometheus exporter for Apache NiFi that collects metrics from the NiFi API and exposes them for Prometheus to scrape.

NiFi API Endpoints Used:

  • /nifi-api/flow/status - Controller metrics
  • /nifi-api/flow/process-groups/root/status - Process group metrics
